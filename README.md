# 📊 E-Commerce Sales & Customer Analytics Dashboard

## 📌 Project Overview

This project is an **end-to-end Data Analytics solution** built using the **Olist Brazilian E-Commerce Public Dataset**. The objective is to transform raw e-commerce data into meaningful business insights by performing data cleaning, data wrangling, feature engineering, exploratory data analysis (EDA), and developing an interactive Power BI dashboard.

The dashboard provides a comprehensive view of sales performance, customer behavior, product performance, seller performance, payment trends, and delivery efficiency, enabling data-driven business decisions.

---

## 🎯 Business Scenario

An e-commerce company wants to monitor its business performance across different regions and product categories. With data spread across multiple tables, stakeholders need a centralized dashboard to track key metrics, identify growth opportunities, understand customer purchasing patterns, and improve operational efficiency.

---

## 🎯 Business Objectives

- Analyze overall sales performance
- Track customer purchasing behavior
- Identify top-performing product categories
- Evaluate seller performance
- Monitor customer satisfaction
- Analyze payment preferences
- Measure delivery performance
- Build an interactive executive dashboard
